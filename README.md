A web application built with **Streamlit** and **MySQL** to manage student records.

## Features
- **Add Students:** Includes validation for age and names.
- **Dashboard:** Analytics for top scorers and pass percentage.
- **Visuals:** Bar and Pie charts.

## How to Run
1. Install libraries: `pip install -r requirements.txt`
2. Setup Database: Run the code in `db.sql` in your MySQL Workbench.
3. Run App: `streamlit run app.py`
